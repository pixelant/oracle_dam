What does it do?
================

This TYPO3 extension allows you to select and import assets from Oracle Content
Management (OCM) Digital Asset Management (DAM) in TYPO3.

Images are downloaded to :file:`fileadmin/user_upload/oracle`.

If an image has changed in OCM, you can update it using the Oracle DAM backend
module.
